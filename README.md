# Calculadora de Calificaciones en C
Este es un proyecto educativo desarrollado por un progrmador de **12 anos**.

## Descripcion
El programa solicita tres notas al usuario, calcula el promedio automaticamente y devuelve el mensaje motivacional basado en el resultado.
Fue desarrollado en **Arch Linux** utilizando GCC.

## Tecnologias
* **Lenjuage:** C
* **Compilador:** GCC
* **Sistema Operativo:** Arch Linux (Kitty Terminal)

## Lo que aprendi
* Uso de 'scanf' para entrada de datos.
* Estructuras de control 'if', 'else if' y 'else'.
* Logica de operadores relacionales y aritmeticos.
* uso de bucles 'while' y 'do-while'.
  
# Calcuador de las tablas de multiplicar en C
Proyecto desarrollado por programador de **12 anos**.
